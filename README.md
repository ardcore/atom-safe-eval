# Colorpicker

Colorpicker package for [atom.io](http://atom.io).

Features:
 - multi-cursor support
 - context detection depending on cursor position
 - simple UI depending on [colorjoe](https://www.npmjs.org/package/colorjoe)

Shortcut: `cmd + alt + k`
